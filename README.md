# ArgentBank FrontEnd

## Getting Started:

### `yarn install`

In order to install all dependencies you need in this project.

Then run :

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
